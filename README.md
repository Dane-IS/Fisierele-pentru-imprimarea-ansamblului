# 🛠️ Fișiere pentru Imprimarea Ansamblului Hardware
### Proiect: Sistem de irigații automatizat IoT

Acest repository conține toate modelele 3D necesare pentru protecția componentelor electronice ale sistemului (Master și Slave).

## 📂 Conținutul pachetului
*   **Carcasă Master:** Protecție pentru ESP32 Gateway și circuitele de alimentare.
*   **Carcasă Slave:** Design compact pentru montare pe ghiveci, cu acces pentru senzori.
*   **Suport Senzori:** Ghidaje pentru higrometru și senzorul de nivel.

## 🖨️ Specificații Recomandate pentru Printare
Pentru a obține o durabilitate optimă în medii cu umiditate ridicată:

*   **Material:** PETG (recomandat) sau PLA.
*   **Infill:** 20% - 30% (Grid/Gyroid).
*   **Strat (Layer height):** 0.2mm.
*   **Suporturi:** Necesare doar pentru zona porturilor USB/Alimentare.

## 🔧 Instrucțiuni de Asamblare
1. Descărcați arhiva `.zip` din acest repository.
2. Utilizați un software de slicing (ex: Cura, PrusaSlicer).
3. După printare, utilizați rășină epoxidică pentru hidroizolarea zonelor de trecere a cablurilor (conform Ghidului DIY).
